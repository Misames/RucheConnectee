# Ruche Connectée

## Objectif

    Réalisez pour un apiculteur une ruche connectée pour la surveillance de celci.

## **Matériels utilisé pour faire le serveur**

    -Capteur de t°/humidité/masse
    -Module GSM
    -Arduinos Mega
    -Raspberry PI 3 (Raspbian OS)
    -Carte SD 8Go
    -Alimentation 5V 2A

## **Langage utilisés**

## **Backend**

    Python
    PHP

## **Frontend**
